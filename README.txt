--------------------------
COMO EXECUTAR O PROJETO :
--------------------------

1. TENHA O PROGRAMA XAMPP OU WAMPP E O MYSQL WORKBENCH INSTALADO NO PC.

2. ABRA O WORKBENCH, CRIE UM BANCO DE DADOS COM O NOME "projetoTelecall" E DÊ UM OK.

3. AO CRIAR O BANCO, COPIE E COLE A QUERY LOCALIZADO NA PÁGINA "banco.sql" DENTRO DO WORKBENCH.

4. EXECUTE A QUERY NO BOTÃO COM UM ICONE DE RAIO.

5. COM O BANCO DE DADOS JÁ CRIADO, ABRA O XAMPP E DÊ UM START NO APACHE E NO MYSQL.

6. MOVA OU COPIE E COLE TODO O PROJETO PARA DENTRO DA PASTA "xampp/htdocs".

7. ABRA SEU NAVEGADOR E E COLE DENTRO DA URL : "http://localhost/Back_End/php/home.php".

8. SE QUISER INSERIR UM USUÁRIO MASTER TESTE, EXECUTE A QUERY LOCALIZADO NA PAGINA "insertMaster.sql".

------------------------------------------------------------------------------------------

DENTRO DESSE PROJETO TEREMOS UMA TELA PRINCIPAL COM DIVERSAS INFORMAÇÕES DA EMPRESA, ALÉM TAMBÉM DA POSSIBILIDADE DE CADASTRO PESSOAL E LOGIN.

NA TELA DE LOGIN TEM A POSSIBILIDADE DO USUÁRIO RECUPERAR SUA SENHA AO CLICAR NO LINK "ESQUECI MINHA SENHA".

ALÉM DE EXISTIR UM SEGUNDO FATOR DE AUTENTICAÇÃO INSERIDO PARA TODOS OS USUÁRIOS, COM 3 PERGUNTAS SECRETAS ALEÁTÓRIAS.

EXISTEM DOIS TIPOS DE USUÁRIOS DENTRO DO SISTEMA, O "COMUM" E O "MASTER", CUJO CADA TIPO DE USUÁRIO TEM ACESSO Á CERTAS FUNÇÕES DENTRO DO SITE.

TEMOS TAMBÉM UM DASHBOARD COM O CADASTRO DE TODOS OS USUÁRIOS COMUNS, SENDO ACESSADO APENAS PELO USUÁRIO MASTER.

O PAINEL DE LOG DÁ ACESSO Á INFORMAÇÃO DE CADA USUÁRIO QUE SE LOGOU NO SISTEMA, ALÉM TAMBÉM DE MOSTRAR SE O USUÁRIO ERROU A PERGUNTA DE AUTENTICAÇÃO.

TANTO O PAINEL DE LOG QUANTO O DASHBOARD TEM A OPÇÃO DE "GERAR PDF".
